TTMKonnect Assets Preview

Logo (logo.svg)
Modern truck logo with connectivity elements. Use this to generate:
  • icon.png (1024x1024) - Main app icon
  • adaptive-icon.png (1024x1024) - Android adaptive icon
  • favicon.png (32x32) - Web favicon

Quick Conversion Steps:

  1. Right-click on the logo above and "Save image as..."
  2. Go to Convertio
  3. Upload the saved SVG file
  4. Set dimensions: 1024x1024 for icons, 32x32 for favicon
  5. Download and replace the PNG files in assets/images/
Splash Screen (splash.svg)
Professional splash screen with dark gradient background. Use this to generate:
  • splash.png (1242x2688) - App launch screen

Quick Conversion Steps:

  1. Right-click on the splash screen above and "Save image as..."
  2. Go to Convertio
  3. Upload the saved SVG file
  4. Set dimensions: 1242x2688
  5. Download and replace splash.png in assets/images/
Next Steps
After generating the PNG files:
  1. Replace the existing PNG files in assets/images/
  2. Restart your development server
  3. Test the app to see the new logo and splash screen
